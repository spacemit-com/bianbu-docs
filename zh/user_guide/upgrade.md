# 系统升级

Bianbu Desktop/NAS 支持在线升级。

## 完全通过命令行

```bash
do-release-upgrade
```

根据提示逐步完成，然后重启。

## 通过命令行打开图形化升级界面(推荐)

仅限 Bianbu Desktop。

```bash
do-release-upgrade -f DistUpgradeViewGtk3
```

## 通过软件更新器

仅限 Bianbu Desktop。

1. 设备联网；
2. 打开“软件更新器”；
3. 等待检查更新完成；
4. 发现新版本，点击升级；
5. 根据提示逐步完成；
6. 重启。
